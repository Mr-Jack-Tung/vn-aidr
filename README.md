# VN AIDr - Open source medical image processing project
## Requirements

- Python 3.7 + Pip
- NodeJS
- Yarn

## 1. Frontend (UI)

### Development

```
cd frontend
yarn
yarn start
```

### Build

```
cd frontend
yarn
yarn build
```

## 2. Server 

```
pip install -r requirements.txt
python app.py
```

- Open `http://localhost:5000` in your browser.
